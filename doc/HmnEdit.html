HmnEdit


概要説明
  • Delphi用エディットコンポーネント
  • 須田さん作のTSsEditをベースに機能追加し新たに作り直しました

インストール方法
  • コンポーネントのインストールで既存もしくは新規パッケージに追加してパッケージをコンパイルして下さい
  • 追加するユニットファイルは、HmnEdit.pasです

アンインストール方法
  • 追加したパッケージからHmnEditとHmnEdit.dcrを削除してパッケージをコンパイルしなおして下さい

THmnEditプロパティ
Alignment: TAlignment default taLeftJustify;
  • テキストの横方向の配置
AutoSize: Boolean default False;
  • デザイン時にテキストの内容に応じてサイズを決定するかどうか
  • 実行時には影響しない
ChangeEventMode: THmnChangeEventMode ;
  • changeイベントの発生のしかた
DisplayDateTimeFormat: string default '';
  • EditOption.EditStyleがesDateとesTimeの時のテキスト編集書式
EditOption: THmnEditOption;
  • 入力後の書式形式等の設定。THmnEditOption参照
ErrColor: TColor default clRed;
  • ShowErrorがTrueのときの表示色
ErrFont: TFont;
  • ShowErrorがTrueのときのフォント（色）
ExitAutoTrim: THmnAutoTrim default eaTrimRight;
  • Exit時に行うTrim設定。全角空白も取り除く対象とする
GoNextOnEnterKey: boolean default False;
  • Enterキー入力で次にフォーカスを移すかどうか
LeftMargins: integer default 0;
  • テキストの左端の表示位置調整
  • AlignmentがtaRightJustify以外の表示位置で効果が出る
MaxValue: string default '';
  • 入力の最大値
MinusColor: TColor default clRed;
  • 入力値がマイナスのときの文字色
  • EditOption.EditStyleがesNumericかesZeroNumかesCommaかesMoneyの時に有効
MinValue: string default '';
  • 入力の最小値
NullValue: string ;
  • テキストが空文字のときにValueプロパティで戻す値
RightMargins: integer default 0;
  • テキストの右端の表示位置調整
  • AlignmentがtaLeftJustify以外の表示位置で効果が出る
ShowEnabledMode: THmnShowEnabledMode default semNormal;
  • EnabledがFalseのときの表示設定
ShowError: boolean default True;
  • エラー表示モードにするかどうか
  • 入力エラーを関知した際には自動でエラー表示モードに切り替える
  • また、エラー表示モードは次の入力で自動的に解除される
ShowMsg: Boolean default True;
  • エラーを関知したときにメッセージ表示を行うかどうか
VAlignment: THmnVAlignment default taTopJustify;
  • テキストの縦方向の配置
Value: Variant;
  • テキストをEditOption.EditStyleに応じて変換した値
  • 代入することも可能
ZeroColor: TColor ;
  • 入力値がゼロのときの文字色
  • EditOption.EditStyleがesNumericかesZeroNumかesCommaかesMoneyの時に有効
ZeroDisplay: boolean default True;
  • 入力値がゼロのとき表示するかどうか
  • EditOption.EditStyleがesNumericかesZeroNumかesCommaかesMoneyの時に有効
Version: string;
  • HmnEditのバージョン

THmnEditイベント
OnChangeAtExit;
  • テキストが変更されてExitが起こったとき
OnEnterKey;
  • Enterキーが押されたとき
OnErrCheck;
  • 入力チェックをするためのイベントハンドラ
  • ただし、このイベントはユーザが開始する必要がある
OnInvalid;
  • 内部でエラーを察知したときに通知されるイベントハンドラ

THmnEditOptionプロパティ
EditStyle: THmnEditStyle default esNone;
  • 入力後の書式編集形式
Length: THmnEditLength;
  • THmnEditLength参照
DoEmptyCheck: Boolean default False;
  • 未入力を許可しないかどうか

THmnEditLengthプロパティ
MaxLength: Integer;
  • 入力できる文字の最大長
DecLength: THmnDecLimit;
  • 小数部の有効桁数。最大5桁
  • EditOption.EditStyleがesCommaかesMoneyの時のみ有効

動作環境
動作環境：Borland Delphi 6、7、8(VCL)
※ただし、D8対応については暫定的な対応です
