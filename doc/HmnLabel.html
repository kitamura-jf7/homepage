HmnLabel


概要説明
  • Delphi用ラベルコンポーネント
  • ラベルにベベル設定を追加し、２分割・３分割表示もできる様にしました

インストール方法
  • コンポーネントのインストールで既存もしくは新規パッケージに追加してパッケージをコンパイルして下さい
  • 追加するユニットファイルは、HmnLabel.pasです

アンインストール方法
  • 追加したパッケージからHmnLabelを削除してパッケージをコンパイルしなおして下さい

THmnLabelプロパティ
Alignment: THmnLBLAlignment default taLeftJustify;
  • キャプションの横方向の配置
BevelInner: TPanelBevel default bvNone;
  • 内部ベベルのスタイルの設定。TPanel参照
BevelOuter: TPanelBevel default bvNone;
  • 外部ベベルのスタイルの設定。TPanel参照
BevelWidth: TBevelWidth default 1;
  • ベベルの幅の設定。TPanel参照
LeftMargins: integer default 0;
  • キャプションの左端の表示位置調整
  • AlignmentがtaRightJustify以外の表示位置で効果が出る
RightMargins: integer default 0;
  • キャプションの右端の表示位置調整
  • AlignmentがtaLeftJustify以外の表示位置で効果が出る
TopMargins: integer default 0;
  • キャプションの上端の表示位置調整
  • LayoutがtlBottom以外の表示位置で効果が出る
BottomMargins: integer default 0;
  • キャプションの下端の表示位置調整
  • LayoutがtlTop以外の表示位置で効果が出る

動作環境
動作環境：Borland Delphi 6、7、8(VCL)
※ただし、D8対応については暫定的な対応です
