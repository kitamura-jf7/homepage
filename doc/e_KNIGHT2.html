The main setting items
Wait
It is "a reservoir" in case a sensor turns up from an end.

Position
Free
It can arrange at the favorite place on a desktop.
Top, Bottom, Left and Right
It sticks to each end of a desktop.

Show
Random illumination
Scan is carried out, changing a Foreground Color at random.
Foreground Color
It is the color of a sensor.
2nd Foreground Color
When colors other than a 'Foreground Color' are specified, scan is carried out, carrying out illuminations between a 'Foreground Color' and the '2nd Foreground Color'.
Background Color
It is a background color when making 3D into a Lowered and Raised.
3D
Either None, Lowered and Raised. When 'None', places other than a block can operate a background another apprications.
Stay on top
It always displays on the front from another apprications.

Lock
A change of a free position and size is made by the ability not doing.

Operation
  • Execution only opens an EXE file.
  • A menu comes out with a right button.
  • A mouse cursor can be united with the Edge of a sensor and size can be changed.
  • The contents of an setting create and save an INI file at the same folder as an EXE file. However, a registry is used when not written out and made to an INI file.
    HKEY_CURRENT_USER\Software\kitamura\KNIGHT2.000

Option of starting
  • Environmental number
    • It specifies with the environmental setting value of eye how many it starts. If there is no specification, it will be used in an order from 1.
    • When having not registered the specified number, an initial-setting value is used.
