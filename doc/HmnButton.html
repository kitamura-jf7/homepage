HmnButton


概要説明
  • Delphi用ボタンコンポーネント
  • ボタン表面の色や文字の色を設定でき、ボタンに縁（フレーム）を付けることができる様にしました

インストール方法
  • コンポーネントのインストールで既存もしくは新規パッケージに追加してパッケージをコンパイルして下さい
  • 追加するユニットファイルは、HmnButton.pasです

アンインストール方法
  • 追加したパッケージからHmnButtonとHmnButton.dcrを削除してパッケージをコンパイルしなおして下さい

THmnButtonプロパティ
MouseIn: Boolean;(Readonly)
  • マウスカーソルがボタンの上にあるかどうか
BevelInner: TPanelBevel default bvNone;
  • 内部ベベルのスタイルの設定。TPanel参照
BevelOuter: TPanelBevel default bvNone;
  • 外部ベベルのスタイルの設定。TPanel参照
BevelWidth: TBevelWidth default 1;
  • ベベルの幅の設定。TPanel参照
CapAlignH: TNHAlignment default haCenter;
  • キャプションの横方向の配置
CapAlignV: TNVAlignment default vaCenter;
  • テキストの縦方向の配置
CapFixed: Boolean default True;
  • フォーカスがあるときキャプションを微調整するかどうか
CapOffsetH: integer default 0;
  • キャプションの横方向の表示位置調整
  • CapAlignHがhaLeftかhaRightのとき効果が出る
CapOffsetV: integer default 0;
  • キャプションの縦方向の表示位置調整
  • CapAlignVがvaTopかvaBottomのとき効果が出る
CaptionLines: TStrings;
  • キャプション。複数行のキャプションをここでセットできる
  • ここにセットされなければ通常のCaptionが使用される
Colors: THmnColors;
  • ボタンの各種色設定。THmnColorsプロパティ参照
Fonts: THmnFonts read FFonts write SetFonts;
  • ボタンの各種フォント設定。THmnFontsプロパティ参照
FrameHeight: integer default 0;
  • 上下に付ける縁（フレーム）の各高さ
FrameWidth: integer default 0;
  • 左右に付ける縁（フレーム）の各幅
ShowFocus: Boolean default True;
  • ボタンにフォーカスがあるときにフォーカス線を表示するかどうか
Style: TButtonStyle default bsStd;
  • ボタンのスタイル
TestShow: THBTestShow default tsNormal;
  • デザイン時の表示モード

THmnButtonイベント
OnMouseEnter;
  • ボタンの上にマウスカーソルが入ってきたとき
OnMouseLeave;
  • ボタンの上からマウスカーソルが出ていったとき

THmnColorsプロパティ
Color: TColor default clBtnFace;
  • ボタンの表面の色
MouseInColor: TColor default clBtnFace;
  • ボタンの上にマウスカーソルがあるときの表面の色
EdgeHighlight: TColor default clBtnHighlight;
  • ボタンエッジの明るい方の色
EdgeShadow: TColor default clBtnShadow;
  • ボタンエッジの暗い方の色
FrameColor: TColor default clBtnFace;
  • ボタンフレームの色
FocusFrameColor: TColor default clWindowFrame;
  • ボタンにフォーカスがあるときのフレームの色
MouseInFrameColor: TColor default clWindowFrame;
  • ボタンの上にマウスカーソルがあるときのフレームの色

THmnFontsプロパティ
Font: TFont default Parent.Font;
  • キャプションのフォント（色）
FocusFont: TFont default Parent.Font;
  • ボタンにフォーカスがあるときのキャプションのフォント（色）
MouseInFont: TFont default Parent.Font;
  • ボタンの上にマウスカーソルがあるときのキャプションのフォント（色）

動作環境
動作環境：Borland Delphi 6、7、8(VCL)
※ただし、D8対応については暫定的な対応です

その他・既知の問題点
  • 縁（フレーム）部分をクリックされた際にボタンのクリックは発生しない様にしていますが、その関係で、ボタンを押し下げたままボタンの外にマウスカーソルを逃がす際にフレームを出るまで押し下がったままになってしまう
